# Usako Game

JavaFXで開発した、うさこを操作する2つのミニゲームを楽しめるアクションゲーム集です。
クラシックな「Flappy Bird」風のゲームと、横スクロールの「ランナーゲーム」が遊べます。

## 収録ゲーム (Game Modes)

### 1. Flappy Usako (フラッピーうさこ)
土管の隙間を縫って飛び続けるゲームです。
- **操作**: `スペースキー` または `上矢印キー` でジャンプ（上昇）
- **ルール**: 土管や地面にぶつかるとゲームオーバー。

### 2. Usako Run! (うさこラン！)
迫りくる障害物を避けて走り続けるランナーゲームです。
- **操作**: 
  - `上矢印キー`: ジャンプ
  - `下矢印キー`: しゃがむ
- **ルール**: 障害物に当たるとゲームオーバー。時間が経つにつれてスピードアップ！

## 機能
- **ランキング機能**: 各ゲームの上位5位までのハイスコアと名前を保存できます。
- **アニメーション**: プレイヤーキャラクターやUIのアニメーション演出。
- **サウンド**: 自作のシンセサイズサウンドによるレトロな効果音（ジャンプ、スコア、ゲームオーバー）。
- **レスポンシブUI**: ゲームオーバー時のオーバーレイ表示やアニメーションボタン。

## ダウンロードとインストール
右の「**Releases**」ページから最新のインストーラーをダウンロードできます。

### Windowsの方
`UsakoGame-x.x.x.exe` をダウンロードして実行してください。
インストールウィザードに従ってインストールすれば、すぐに遊べます。

### Macの方
`UsakoGame-x.x.x.dmg` をダウンロードしてください。
1. ファイルを開き、うさこアイコンを「Applications」フォルダにドラッグ＆ドロップします。
2. **初回起動時の注意**: 
   開発者署名を行っていないため、ダブルクリックでは「開けません」という警告が出ます。  
   OSバージョンによってアプリを開く方法が異なります。

   **方法A**:
   アプリアイコンを**右クリック（またはControl+クリック）して「開く」**を選び、確認画面で「開く」を押してください。

   **方法B（方法Aで開けない場合）**:
   1. 一度ダイアログの「完了」を押して閉じます。
   2. 「システム設定」→「プライバシーとセキュリティ」を開きます。
   3. 下の方にスクロールすると「セキュリティ」の項目に「UsakoGameは開発元を確認できないため...」という表示があります。
   4. **「このまま開く」**ボタンを押して、認証してください。

   （2回目からは普通に起動できるようになります）

## セーブデータについて
- **保存場所**: 誤って削除されないよう、OS標準の安全な場所に保存されます。
  - Windows: `%APPDATA%\UsakoGame`
  - Mac: `~/Library/Application Support/UsakoGame`
- **データの削除**:
  ゲーム内の「ランキング」画面にある「ランキングデータを全削除」ボタンから、いつでも全てのデータを消去できます。

## 開発者向け実行方法 (For Developers)
ソースコードから直接実行する場合は、JDK 23以降が必要です。
プロジェクトのルートディレクトリで以下のコマンドを実行してください。

```bash
./mvnw clean javafx:run
```

## リリース（配布）の手順 (How to Release)

GitHub Actionsを設定済みのため、タグをプッシュすると自動的にインストーラーが作成され、GitHubのReleasesページに公開されます。

### 1. タグを付与してプッシュ
バージョン番号のタグ（例: `v1.0.2`）を作成し、GitHubへプッシュします。

```bash
git tag v1.0.2
git push origin v1.0.2
```

### 2. リリースの確認
数分待つと、GitHubの [Releasesページ](https://github.com/RiTa-23/PakuPakuBird/releases) に自動的にドラフト（または公開）リリースが作成され、Windows/Mac用のインストーラーがアップロードされます。
